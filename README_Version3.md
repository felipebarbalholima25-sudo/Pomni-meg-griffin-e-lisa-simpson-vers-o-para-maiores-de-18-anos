# Mona Alberta — Protótipo Ren'Py

Aviso importante
- Este protótipo é destinado a um público adulto. Confirme que todas as personagens são maiores de 18 anos antes de usar/compartilhar.
- Não inclua conteúdo envolvendo menores.

O que há aqui
- Estrutura mínima de um projeto Ren'Py em `game/`:
  - `game/script.rpy` — script principal com diálogos, escolhas e rotas.
  - `game/options.rpy` — configurações básicas.
  - `game/images/` — pasta para colocar as imagens (placeholders listados em `game/README_ASSETS.txt`).

Como rodar localmente
1. Instale Ren'Py: https://www.renpy.org/latest.html
2. Crie um novo projeto no launcher Ren'Py ou copie esta pasta `game/` para a pasta `renpy-<versão>/projects/<nome_do_projeto>/game/`.
3. Substitua as imagens em `game/images/` pelos seus assets (mesmos nomes ou ajuste `script.rpy`).
4. Abra o Ren'Py Launcher e execute o projeto.
